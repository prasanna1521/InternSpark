File Automation Project

Automate your file management tasks with ease.

About the Project

This Python script automates file operations like renaming, sorting, and cleaning folders.

Usage

python automation.py

Example

Enter a directory path and choose an operation:

  1. Rename Files
  2. Sort Files
  3. Clean Empty Folders

Sample Output

Renamed: file1.txt → report1.txt
Moved: image.png → Images/

✔ Files renamed successfully.

❌ Error: Directory not found.

⚠ Invalid choice. Please try again.
