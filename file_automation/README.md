# File Organizer

## Objective
This project is a Python automation script that organizes files into folders based on their file extensions.

## Features
- Organizes files automatically
- Uses the OS module
- Supports user input
- Generates operation logs
- Uses exception handling

## Technologies Used
- Python 3
- os module
- shutil module
- logging module

## How to Run
1. Open the project in VS Code.
2. Run:
   python organizer.py
3. Enter the folder path when prompted.
4. The files will be organized automatically.

## Project Structure

Task-1-File-Organizer/
│── organizer.py
│── operations.log
│── README.md
└── test_files/

## Output
Files are sorted into folders such as:
- Images
- Documents
- Music
- Videos
